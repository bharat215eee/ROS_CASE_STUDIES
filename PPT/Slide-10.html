Troubleshooting

ModuleNotFoundError

ImportError: No module named 'case_study3_...'

Fix: You likely forgot to source the setup file after building. Run:

source ~/catkin_ws/devel/setup.bash

No Motion Visible

The server says "Completed" but the turtle didn't move.

  • Verify /turtle1/cmd_vel is correct.
  • Check rosparam get /motion/linear_speed (might be 0).

Service Hangs

Client waits indefinitely.

Fix: Ensure motion_server.py is actually running. The client uses wait_for_service() which blocks until the server is available.

Pro Tip

Use rosservice call /reset to clear the turtlesim screen and move the turtle back to the center between tests.
