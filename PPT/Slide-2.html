ROS Services & Parameter Server

Case Study-3: Turtlesim

Build a custom service to control Turtlesim behavior dynamically using the ROS Parameter Server.

Robotics Operating System Tutorial

Objectives

Create RunMotion.srv and generate custom messages.

Implement robust Python server and client nodes.

Utilize the Parameter Server to tune behavior at runtime.

Execute end-to-end tests with verification.

Prerequisites

ROS Noetic/Melodic Turtlesim Python 3 Catkin

Expected Outcome

A functional system where the /run_motion service controls the /turtle1/cmd_vel topic using three distinct motion modes: line, circle, and square.
