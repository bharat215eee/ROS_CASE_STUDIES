Create the standard Catkin directory structure. Use catkin_make to generate the build files and add the setup script to your bashrc for convenience.

Build & Source

Compile the workspace again to register the new package and source the development environment overlay.

Tip: Always source setup.bash after adding new packages.

user@ros-machine: ~/catkin_ws

# 1. Setup Workspace

mkdir -p ~/catkin_ws/src

cd ~/catkin_ws

catkin_make

echo "source ~/catkin_ws/devel/setup.bash" >> ~/.bashrc

source ~/.bashrc

# 2. Create Package

cd ~/catkin_ws/src

catkin_create_pkg case_study3_turtlesim_service rospy std_msgs geometry_msgs turtlesim

# 3. Build & Source

cd ~/catkin_ws

catkin_make

source devel/setup.bash
