ROS Topic Model

// ROS Noetic Topic Abstraction Node = process that performs computation Topic = named bus over which nodes exchange messages Message = strongly-typed data structure Examples (TurtleSim): /turtle1/cmd_vel : geometry_msgs/Twist (velocity commands) /turtle1/pose : turtlesim/Pose (x, y, θ, v, ω) // Conceptual I/O mapping Publisher Node ---> Topic ---> Subscriber Node u(t) ---> y(t)
Node: ROS process
Topic: Named data stream
Message: Typed data packet

Publisher–Subscriber Flow

// Example: /chatter string topic talker.py → publishes std_msgs/String on /chatter listener.py → subscribes to /chatter and prints data Node Graph (simplified): [talker] --(std_msgs/String)--> [/chatter] --(std_msgs/String)--> [listener] You can inspect this structure with: rqt_graph rostopic list rostopic info /chatter
Rate: Publication frequency (Hz)
Queue size: Subscriber buffer
ROS Master: Name service & registration

talker.py – String Publisher

ROS Noetic Python Python 
#!/usr/bin/env python3
import rospy
from std_msgs.msg import String

def talker():
    pub = rospy.Publisher('/chatter', String, queue_size=10)
    rospy.init_node('talker', anonymous=True)
    rate = rospy.Rate(10)  # 10 Hz
    count = 0

    while not rospy.is_shutdown():
        msg = String()
        msg.data = f"Hello ROS Noetic #{count}"
        pub.publish(msg)
        rate.sleep()
        count += 1

if __name__ == '__main__':
    try:
        talker()
    except rospy.ROSInterruptException:
        pass

listener.py – String Subscriber

ROS Noetic Python Python 
#!/usr/bin/env python3
import rospy
from std_msgs.msg import String

def callback(msg: String):
    rospy.loginfo("I heard: %s", msg.data)

def listener():
    rospy.init_node('listener', anonymous=True)
    rospy.Subscriber('/chatter', String, callback)
    rospy.spin()

if __name__ == '__main__':
    listener()

signal_publisher.py – Numeric Topic

ROS Noetic Python Python 
#!/usr/bin/env python3
import rospy
import math
from std_msgs.msg import Float64

def signal_pub():
    pub = rospy.Publisher('/signal', Float64, queue_size=10)
    rospy.init_node('signal_publisher')
    rate_hz = rospy.get_param('~rate', 50.0)
    freq = rospy.get_param('~freq', 1.0)  # Hz
    rate = rospy.Rate(rate_hz)
    t0 = rospy.Time.now().to_sec()

    while not rospy.is_shutdown():
        t = rospy.Time.now().to_sec() - t0
        y = math.sin(2.0 * math.pi * freq * t)
        pub.publish(Float64(y))
        rate.sleep()

if __name__ == '__main__':
    signal_pub()
