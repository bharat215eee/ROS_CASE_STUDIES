Service Abstraction

// ROS Service = request–response interaction Topic : continuous, streaming data (publish/subscribe) Service : one-time RPC-style call (client/server) Service type: <request fields> --- <response fields> Example (turtlesim /spawn): float32 x float32 y float32 theta string name --- string name Calling /spawn once creates a new turtle at (x, y, theta) and returns its name.
Client: Sends request
Server: Provides response
Tool: rosservice

E3 – Custom Service: Distance Between Turtles

// Objective Create a custom service that returns the Euclidean distance between two turtles (turtle1 and turtle2). // Outline - Service type: Empty request → response float64 distance (for simplicity). - Server node: - Subscribes to both /turtle1/pose and /turtle2/pose - On request, computes: d = sqrt((x2 - x1)² + (y2 - y1)²) - Client node: - Calls service and prints distance. // Analysis Compare this to computing state error in your control labs (distance to reference trajectory).

distance_srv_server.py – Service Server

ROS Noetic Python Python 
#!/usr/bin/env python3
import rospy
import math
from turtlesim.msg import Pose
from std_srvs.srv import Trigger, TriggerResponse

pose1 = Pose()
pose2 = Pose()

def cb_t1(msg):
    global pose1
    pose1 = msg

def cb_t2(msg):
    global pose2
    pose2 = msg

def handle_distance(req):
    dx = pose2.x - pose1.x
    dy = pose2.y - pose1.y
    d  = math.sqrt(dx*dx + dy*dy)
    resp = TriggerResponse()
    resp.success = True
    resp.message = f"Distance between turtles: {d:.3f}"
    return resp

def main():
    rospy.init_node('distance_srv_server')
    rospy.Subscriber('/turtle1/pose', Pose, cb_t1)
    rospy.Subscriber('/turtle2/pose', Pose, cb_t2)
    srv = rospy.Service('/turtles_distance', Trigger, handle_distance)
    rospy.loginfo('Distance service ready: /turtles_distance')
    rospy.spin()

if __name__ == '__main__':
    try:
        main()
    except rospy.ROSInterruptException:
        pass

distance_srv_client.py – Service Client

ROS Noetic Python Python 
#!/usr/bin/env python3
import rospy
from std_srvs.srv import Trigger

def main():
    rospy.init_node('distance_srv_client')
    rospy.wait_for_service('/turtles_distance')
    srv = rospy.ServiceProxy('/turtles_distance', Trigger)

    try:
        resp = srv()
        rospy.loginfo(resp.message)
    except rospy.ServiceException as e:
        rospy.logerr("Service call failed: %s", e)

if __name__ == '__main__':
    main()
